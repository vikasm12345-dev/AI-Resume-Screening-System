# 🤖 AI Resume Screening System

An **intelligent AI-based Resume Screening System** that automatically matches multiple resumes against a job description using **Natural Language Processing (NLP)**, **TF-IDF vectorization**, and **cosine similarity** algorithms.

## 📌 Overview

This project is a real-world **AI-powered hiring tool** that automates the resume screening process. HR teams can upload multiple resumes and instantly get ranked candidates based on how closely they match the job requirements.

---

## ✨ Features

- 📄 **Multi-format Support** — Upload PDF, DOCX, or TXT resumes
- 🧠 **NLP Processing** — Text cleaning, tokenization, and lemmatization
- 🔍 **Smart Skill Extraction** — Detects 100+ skills across 6 categories
- 📊 **TF-IDF Similarity** — Advanced content matching algorithm
- ⚖️ **Weighted Scoring** — Customizable content vs skill weights
- 🏆 **Intelligent Ranking** — Sorts resumes by relevance
- 📈 **Interactive Dashboard** — Beautiful charts and metrics
- 📥 **CSV Export** — Download detailed rankings report
- 🎨 **Modern Dark UI** — Professional user interface
